Technical Projects

CHRONOS: Hardware Implementation

The objective of the Cloud-based Hybrid RF-Optical Network Over Synchronization (CHRONOS) project is to design a heterogeneous, hybrid, and synchronous CRAN. By integrating radio frequency and optical technologies, implementing a baseband transceiver on a multi-FPGA platform, and developing advanced edge nodes, the system aims to achieve low latency, high speed, and high bandwidth. The CHRONOS architecture comprises the Hybrid Baseband Processor (HBP), Heterogeneous Network Edge (HNE), and Heterogeneous Mobile Terminal (HTM). The HBP and HNE correspond to the REC and RE, respectively, and are constructed using FPGAs and CPUs. In this study, the HBP and HNE are designed and implemented using two ZYNQ Ultrascale+ RFSoC ZCU111 evaluation boards.

Design & Implementation of a Non-linear Dynamics Based Controller for a 5 DOF Robot

In this project, a dynamics-based state-feedback non-linear controller was designed and implemented for an industrial 5 DOF XR4 Rhino Robot Manipulator. The objective was to compensate the robot’s coupled non-linear dynamics by accurately modeling the robot and carefully designing and implementing a dynamics-based state-feedback non-linear controller. After successfully completing the modeling and controller designing stages, a simulation-based verification was carried out. Then, the verified non-linear controller was successfully implemented in a hardware setup using an ATMEGA based micro-controller together with an interactive graphical user interface (MATLAB).
